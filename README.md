# react-native-ui-bundle

A lightweight React Native library for essential UI components.

## Installation

```sh
npm install react-native-ui-bundle
```

## Usage


```js
import { multiply } from 'react-native-ui-bundle';

// ...

const result = await multiply(3, 7);
```
